---
name: cron-next
description: Compute the next fire time of a 5-field cron expression.
---

Implement `next_run(expr, after)`: the next `datetime` strictly after `after`
(minute resolution, seconds zero) that a 5-field cron expression matches.

Fields: minute hour day-of-month month day-of-week. Parse each into an allowed set,
handling `*`, comma lists, `a-b` ranges, and `*/n` / `a-b/n` steps. Day-of-week uses
Sunday=0 (7 also Sunday); map Python's weekday accordingly.

Rule: if BOTH day-of-month and day-of-week are restricted, match on EITHER. If only
one is restricted, apply only that one. Step minute-by-minute from `after`+1 until a
match (cap the search at a few years for unsatisfiable exprs).

Return only the complete Python source. No fences, no prose.
