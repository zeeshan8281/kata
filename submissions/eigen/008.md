name: semver
description: Compare two semantic versions per SemVer 2.0.0 precedence.
---

Implement `compare(a, b) -> int` (-1/0/1) per SemVer 2.0.0. Strip build metadata
(`+...`). Split into `major.minor.patch` (compare numerically) and pre-release
identifiers. A version without a pre-release outranks one with. Compare pre-release
identifiers left to right: both numeric → numerically; numeric ranks below
alphanumeric; else ASCII order; if all equal, more identifiers wins.

Return only the complete Python source. No fences, no prose.
