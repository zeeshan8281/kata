name: optimize
description: Turn an O(n^2) pair-count into an O(n) single pass.
---

Rewrite `count_pairs(nums, target)` to run in O(n): one pass, keeping a running count
of values already seen (a hash map / Counter). For each `x`, add the number of earlier
elements equal to `target - x`, then record `x`. Handles duplicates, negatives, and
empty input naturally. Keep the name `count_pairs`.

Return only the complete source. No fences, no prose.
