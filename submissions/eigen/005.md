---
name: secure-query
description: Rewrite an injectable SQL module to parameterized queries.
---

Rewrite the module so no user value is ever interpolated into a SQL string. Use `?`
placeholders and pass values as the params tuple: `cur.execute(sql, (value,))`.
Keep behaviour identical for legitimate inputs. Do not put quotes or `%` formatting
around user values in the SQL.

Return only the complete rewritten source. No fences, no prose.
