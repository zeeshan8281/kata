---
name: task-pool
description: Run async tasks with bounded concurrency, results in input order.
---

Implement `async def run_pool(tasks, limit)`. `tasks` is a list of zero-arg async
functions. Gate each with an `asyncio.Semaphore(limit)` so at most `limit` run at once,
and return results in input order — wrap each task with the semaphore and
`asyncio.gather` them (gather preserves input order).

Return only the complete Python source. No fences, no prose.
